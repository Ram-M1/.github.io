# FOCUS PWA

Приложение для саморазвития: 6 сфер жизни + Оракул + Чаты + Привычки-марафоны + Профиль + Кабинет коуча.

## Точка входа
`landing.html` — стартовый экран (регистрация / вход).

## Как залить на GitHub Pages
1. Залей все файлы в КОРЕНЬ репозитория `ЛОГИН.github.io`
2. Settings → Pages → Source: main branch → Save
3. Открой `https://ЛОГИН.github.io/landing.html`

## ⚡ ПРОВЕРКА FIREBASE (сделать первым делом)
Регистрация и вход теперь работают через реальный Firebase (проект focus-21230).

**Что проверить:**
1. Залей файлы на GitHub (обязательно https — Firebase работает только по https, GitHub Pages это даёт)
2. Открой `register.html` на телефоне
3. Зарегистрируйся (email + пароль)
4. Должен быть переход на profile-form.html
5. Зайди в Firebase Console → Authentication → Users — там должен появиться твой email
6. Проверь вход (login.html) тем же email/паролем

**Если регистрация не работает** — смотри консоль браузера (ошибки), проверь что firebase-config.js и firebase-auth-helper.js залиты.

## Ключевые файлы
- `firebase-config.js` — конфиг Firebase (инициализация)
- `firebase-auth-helper.js` — регистрация/вход/выход
- `focus-storage.js` — единое хранилище данных (точка перехода на Firebase)
- `focus-core.js` — анимации, темы, celebrate
- `focus-analytics.js` — аналитика (спит до заполнения ID)

## Входы в скрытые разделы
- **Кабинет коуча:** клик на логотип в index → меню
- **Админка:** долгое нажатие (1.5с) на имя в профиле
