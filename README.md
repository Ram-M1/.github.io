# FOCUS — экосистема саморазвития (PWA)

Точка входа: **landing.html**

## Заливка на GitHub Pages
1. Репозиторий: `ТВОЙЛОГИН.github.io` (только латиница/цифры/дефис)
2. Загрузи ВСЁ содержимое этой папки в КОРЕНЬ репозитория
3. Settings → Pages → ветка main, папка / (root) → Save
4. Откроется на `https://ТВОЙЛОГИН.github.io`

## Обновление на айфоне (кэш упрямый!)
После заливки: потяни страницу вниз для обновления / закрой-открой вкладку Safari / при необходимости удали иконку PWA и очисти данные сайта в Настройки → Safari.

## Что внутри (53 экрана)
Онбординг, главная (6 сфер + ИИ-наставник), все сферы с подразделами, оракул,
чаты (группы, голосовые, перевод F-coin), родительский контроль (карта, геозоны),
B2B кабинет коуча (программы, домашки, статистика, рейтинг), профиль, админка.

## Данные
Сейчас в localStorage устройства. Полноценная работа (синхронизация, реальный чат,
ИИ, оплата, связь коуч-клиент и родитель-ребёнок) — после подключения бэкенда.
