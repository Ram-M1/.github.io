# FOCUS — экосистема саморазвития (PWA)

Точка входа: **landing.html**

## Запуск на GitHub Pages
1. Создай репозиторий с именем `ТВОЙЛОГИН.github.io` (только латиница, цифры, дефис — без кириллицы и пробелов)
2. Загрузи ВСЕ файлы из этой папки в корень репозитория
3. Settings → Pages → Source: ветка main, папка / (root) → Save
4. Через пару минут приложение откроется по адресу `https://ТВОЙЛОГИН.github.io`

## Что внутри (52 экрана)
- Онбординг: landing → register → confirm → profile-form → index (вход: login)
- Главная: 6 сфер + ИИ-блок
- Тело, Менталка, Энергия, Отношения, Вера (со всеми подразделами)
- Оракул: каталог, заявка, кабинет (oracle_menu, psychic_apply, psychic_dashboard)
- Чаты: список, переписка, плавающая кнопка (fokus_chats, fokus_chat, focus-chat-fab.js)
- Админка: admin.html (вход — долгое нажатие на имя в профиле)
- Профиль: О себе, Галерея, Работа/бизнес, Финансы, Friend Code, Уведомления, Магазин
- Родительский контроль: parental_control, control_child, location_map (карта с геозонами)

## Важно
- Данные сейчас в localStorage (для первых пользователей нормально)
- ИИ, реальный чат, оплата, распознавание чеков, коннект семьи/детей — подключаются на этапе бэкенда (Firebase + платежи + AI)
- 5 тем оформления переключаются в приложении
