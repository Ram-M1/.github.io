# FOCUS PWA
Точка входа: landing.html
Firebase: focus-21230 (auth + Firestore + автосинк профиля)

## ВАЖНО про кэш (для разработки)
Service worker теперь network-first для JS/CSS — свежие файлы подтягиваются
автоматически при обновлении страницы. Переустанавливать PWA больше не нужно.
При заливке новой версии: просто обнови страницу (или закрой-открой вкладку).

## Родительский контроль
- Родитель: Родительский контроль → Добавить ребёнка → код
- Ребёнок: Профиль → долгий тап на имя (1.5с) → ввод кода
- Ребёнок появляется у родителя автоматически после привязки
